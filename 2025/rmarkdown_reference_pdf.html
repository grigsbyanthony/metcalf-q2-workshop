YAML Header

---
title: "Document Title"
author: "Your Name"
date: "2025-06-05"
output: html_document
---

Output Options:

  • html_document - Web page
  • pdf_document - PDF
  • word_document - Word doc
  • github_document - GitHub

Code Chunks

Basic Chunk:

```{r}
# R code here
summary(data)
plot(x, y)
```

With Options:

```{r name, echo=FALSE, eval=TRUE}
# Code here
```

Key Chunk Options:

OptionEffect
echo=FALSEHide code
eval=FALSEDon't run code
include=FALSEHide everything
message=FALSEHide messages
warning=FALSEHide warnings
fig.width=8Plot width
cache=TRUESave results

Global Setup

```{r setup, include=FALSE}
knitr::opts_chunk$set(
  echo = TRUE,
  warning = FALSE,
  message = FALSE,
  fig.width = 8
)
```

Useful Functions

# Pretty tables
knitr::kable(data)
DT::datatable(data)

# Include images
knitr::include_graphics("img.png")

Keyboard Shortcuts:

Ctrl/Cmd + Shift + K - Knit document
Ctrl/Cmd + Alt + I - Insert chunk
Ctrl/Cmd + Enter - Run line/selection
Pro Tips:
• Use descriptive chunk names
• Set global options in setup chunk
• Cache long computations
• Use relative paths for portability

Special Formatting

Superscript: R^2^
Subscript: H~2~O
Footnote: text^[footnote]
